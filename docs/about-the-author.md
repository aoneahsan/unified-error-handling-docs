## The project

The library standardizes error handling across web and mobile projects so teams do not rebuild error normalization, context/breadcrumb handling, adapter configuration, and framework glue for every new product. Pick the service that fits the project; the capture code stays the same. It is MIT-licensed and open to contributions.

- npm package: [`unified-error-handling`](https://www.npmjs.com/package/unified-error-handling)
- Source: [github.com/aoneahsan/unified-error-handling](https://github.com/aoneahsan/unified-error-handling)
